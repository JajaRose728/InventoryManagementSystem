🐛 QUICK TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════════

  Port already in use?
  → taskkill /PID <PID> /F (Windows)
  → lsof -i :<port> && kill -9 <PID> (Mac/Linux)

  Emulator won't start?
  → java -version (check Java installed)
  → firebase --version (check Firebase tools)

  Backend can't connect?
  → Verify emulator is running (http://localhost:4000)
  → Check .env has FIRESTORE_EMULATOR_HOST=localhost:8080

  Frontend shows errors?
  → Check browser console (F12)
  → Verify backend is running (curl http://localhost:3000/health)
